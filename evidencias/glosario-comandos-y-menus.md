# Glosario de comandos, menús y herramientas

Completa o estudia este glosario durante la actividad.

| Término | Definición | Ejemplo de uso |
|---|---|---|
| VS Code | Editor de código usado para crear y revisar proyectos. | Abrir la carpeta del proyecto. |
| Explorer / Explorador | Panel donde se ven carpetas y archivos. | Verificar si existe `css/estilos.css`. |
| Open Folder / Abrir carpeta | Opción para abrir un proyecto completo. | Abrir `detectives-error-web-rural-v2`. |
| Save / Guardar | Acción para conservar cambios en un archivo. | Guardar después de corregir una ruta. |
| Terminal | Panel para ejecutar comandos. | Abrir una carpeta o ejecutar herramientas. |
| Run / Ejecutar | Acción para iniciar un programa. | Abrir `index.html` en el navegador. |
| Debug / Depurar | Revisar el funcionamiento paso a paso. | Buscar dónde falla un script. |
| Problems / Problemas | Panel que muestra advertencias o errores del editor. | Detectar errores de escritura. |
| Console / Consola | Herramienta del navegador que muestra errores y mensajes. | Ver errores de JavaScript. |
| Inspector | Herramienta para revisar HTML y CSS en el navegador. | Ver si un estilo se aplicó. |
| Network / Red | Pestaña que muestra archivos cargados o no cargados. | Detectar un CSS con ruta incorrecta. |
| Breakpoint | Punto donde se detiene el programa durante depuración. | Pausar una función para ver variables. |
| Path / Ruta | Ubicación de un archivo. | `css/estilos.css`. |
| Relative path / Ruta relativa | Ruta escrita desde la ubicación del archivo actual. | `js/app.js`. |
| `document.getElementById()` | Instrucción para buscar un elemento HTML por su id. | `document.getElementById("fecha")`. |
| `Number()` | Convierte un texto numérico a número. | `Number("36")`. |
| `localStorage` | Almacenamiento del navegador. | Guardar registros aunque se recargue la página. |
| `addEventListener()` | Permite escuchar eventos del usuario. | Ejecutar una función al presionar un botón. |
| `submit` | Evento que ocurre al enviar un formulario. | Registrar datos de producción. |
| `innerHTML` | Permite insertar contenido HTML dentro de un elemento. | Crear filas de una tabla. |
