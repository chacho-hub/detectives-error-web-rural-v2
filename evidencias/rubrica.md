# Rúbrica de evaluación

| Criterio | Bajo | Básico | Alto | Superior |
|---|---|---|---|---|
| Identificación de errores | Encuentra pocos errores o no los clasifica. | Encuentra algunos errores básicos. | Encuentra la mayoría y los clasifica. | Encuentra, clasifica y explica todos con claridad. |
| Uso de herramientas | No usa consola ni inspector. | Usa una herramienta con ayuda. | Usa consola, inspector y VS Code con autonomía. | Usa varias herramientas y justifica su utilidad. |
| Corrección del código | El proyecto sigue sin funcionar. | Corrige parcialmente. | El proyecto funciona correctamente. | El proyecto funciona y además incluye mejoras. |
| Pruebas | No prueba con datos reales. | Prueba con pocos datos. | Usa los casos de prueba indicados. | Usa casos de prueba y propone pruebas adicionales. |
| Bitácora | Incompleta o superficial. | Registra algunos errores. | Registra errores, causas y soluciones. | Documenta con detalle y reflexión técnica. |
| Micro-quiz y glosario | No entregado. | Respuestas incompletas. | Respuestas claras. | Respuestas claras, ejemplos propios y vocabulario técnico. |
| Retos adicionales | No realiza retos. | Realiza un reto simple. | Realiza dos retos funcionales. | Realiza varios retos con creatividad y buena presentación. |
